# Videos M14 Studio

## Hero Background Video

**Archivo necesario:** `hero-reel.mp4`

**Especificaciones:**
- Formato: MP4 (H.264)
- Resolución: 1920x1080 mínimo (4K recomendado)
- Duración: 10-20 segundos (loop)
- FPS: 30 o 60
- Peso: Optimizado para web (máx 10-15MB)

**Características del video:**
- Cinematográfico y premium
- No distractivo (el contenido debe poder leerse encima)
- Color grading oscuro/neutral
- Sin audio (el sitio usa autoplay muted)
